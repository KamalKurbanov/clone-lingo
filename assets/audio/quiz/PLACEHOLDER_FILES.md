## Требования к аудиофайлам
- **Формат**: MP3
- **Качество**: 128-320 kbps
- **Частота**: 44.1 kHz
- **Длительность**: 1-3 секунды на слово
- **Размер**: < 50KB на файл
- **Язык**: Английский (или целевой язык обучения)

## Как создать аудиофайлы
1. Используйте текстово-речевые сервисы (TTS)
2. Запишите голос профессионального диктора
3. Используйте онлайн инструменты для создания аудио
4. Обработайте файлы для оптимизации размера

## Рекомендуемые TTS сервисы
- Google Text-to-Speech
- Amazon Polly
- Microsoft Azure Speech Service
- Natural Reader
- iSpeech

## Структура именования
```
[тип-викторины]-[слово].mp3
```
Примеры:
- `audio-quiz-apple.mp3`
- `image-quiz-house.mp3`
- `mixed-quiz-cat.mp3`

## Примечания
- Убедитесь, что произношение четкое и понятное
- Используйте нейтральный акцент для международного понимания
- Тестируйте качество звука на разных устройствах
- Создавайте резервные копии оригинальных файлов
